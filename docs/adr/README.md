# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for ClipCal.

ADRs document significant architectural choices: the context that made them necessary, the decision made, the alternatives considered, and the consequences. Status moves from `PROPOSED` (in PR) to `ACCEPTED` (after merge) to `SUPERSEDED` (if replaced).
